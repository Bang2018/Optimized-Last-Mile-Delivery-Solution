# Optimized-Last-Mile-Delivery-Solution

The project under development aims to revolutionize the last mile delivery process, leveraging advanced technologies from Amazon Web Services (AWS). This innovative solution is designed to efficiently address the challenges faced by up to 50 customers within a remarkable average time of 33 seconds, with zero rejected deliveries. The system also demonstrates the capacity to manage deliveries for 200 customers in under a minute.

Currently in the development stage, this project is addressing a two-stage optimization problem. To ensure optimal delivery routes and minimize delivery rejections, it employs sophisticated algorithms such as Iterated Local Search (ILS) and Ant-Colony Optimization (ACO). These algorithms are crucial in rerouting deliveries effectively, reducing the likelihood of rejected deliveries by approximately 30%.

The project's primary objective is to streamline the last mile delivery process, enhancing efficiency, reducing costs, and improving customer satisfaction. By optimizing routes and minimizing delivery rejections, the solution promises significant benefits for both logistics providers and end customers alike. We look forward to the successful completion of this project, which has the potential to redefine the modern approach to last mile delivery.
